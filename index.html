Cómo está pensada la herramienta

La biblia del proyecto está oculta dentro del código. Aquí solo trabajas con la memoria narrativa útil: sinopsis de capítulos, notas de continuidad y, si quieres, el texto completo.

El prompt de sinopsis sirve para pedir primero la propuesta del nuevo capítulo. El prompt de redacción sirve para escribirlo completo una vez ya tengas aprobada la sinopsis.
